▶️ Cómo ejecutar el prototipo en tu PC

Requisitos previos: Node.js

* Instala las dependencias:

* npm install


Configura tu clave de Gemini:
en el archivo .env.local, establece el valor de GEMINI_API_KEY con tu API Key de Gemini.

Ejecuta la app:

* npm run dev
